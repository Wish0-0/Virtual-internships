This project was completed as part of the ShadowFox Cybersecurity Virtual Internship using a virtual machine-based lab environment with Kali Linux and Windows 10. The internship focused on practical cybersecurity concepts including network reconnaissance, directory enumeration, packet analysis, password auditing, executable analysis, reverse shell connections, and wireless security testing. Various industry-standard tools such as Nmap, Gobuster, Wireshark, John the Ripper, Metasploit Framework, PE Explorer, VeraCrypt, and Aircrack-ng were used to perform hands-on security assessments in a controlled and ethical environment. The project helped strengthen practical knowledge in penetration testing, digital forensics, network security, and vulnerability assessment while providing real-world exposure to cybersecurity techniques and defensive security practices.
